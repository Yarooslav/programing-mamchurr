Визначити, чи є серед цифр заданого трьох-значного числа однакові цифри.


Основна частина:
- опис роботи основної функції: знайти однакові цифри  int  number = 122;int num1 = number / 100 % 10;int num2 = number / 10 % 10;int num3 = number % 10;if (num1 == num2 || num2 == num3 || num3 == num1) printf("Одинаковые цифры есть");else printf("Одинаковых цифр нет");


перелік вхідних даних:

   - number - будь-яке трьохзначне число (int).



- дослідження результатів роботи програми:

 int  number = 122;//вводимо будь-яке трьозначне число 
 int num1 = number / 100 % 10;//создаем переменую num 1. В ней наше число (number) делим на 100 и ещё раз на 10 = 122 / 100 % 10 
 int num2 = number / 10 % 10;//создаем переменую num 2. В ней наше число (number) делим на 10 и ещё раз на 10 = 122 / 10 % 10 
 int num3 = number % 10;//создаем переменую num 3. В ней наше число (number) делим на 10 = 122 / 10 % 10
 
if (num1 == num2 || num2 == num3 || num3 == num1) printf("Одинаковые цифры есть");//здесь мы  проверяем ,если ли одинаковые цифры 
else printf("Одинаковых цифр нет");
